# Grok Business Guide

Guide for using Grok Business and Grok Business Enterprise.

## Overview

Grok Business provides enterprise-grade access to Grok through grok.com with additional features for teams and organizations.

## Workspaces

### Switching Workspaces

Use the workspace selector in the bottom left navigation on grok.com to switch between workspaces.

### Workspace Types

- **Personal Workspace**: Your individual workspace
- **Team Workspace**: Shared workspace for team collaboration
- **Organization Workspace**: Enterprise workspace with admin controls

## Features

### Chat Interface

Access Grok through the web interface at grok.com:
- Multi-turn conversations
- File uploads
- Image understanding
- Web search integration
- X search integration

### Conversation Management

- Save conversations
- Share conversations with team members
- Organize conversations into folders
- Search conversation history

### File Handling

Upload and analyze documents directly in chat:
- PDF documents
- Text files
- Images
- Spreadsheets

## Team Collaboration

### Sharing

- Share conversations with team members
- Collaborate on documents
- Team-wide prompt templates

### Permissions

Control who can:
- View conversations
- Edit shared resources
- Invite new members

## Enterprise Features

### Security

- SSO (Single Sign-On) integration
- SAML authentication
- Role-based access control
- Audit logging

### Compliance

- Data residency options
- Compliance certifications
- Data retention policies

### Administration

- User management
- Usage analytics
- Billing management
- Policy enforcement

## API Access

Grok Business users can also access the API:
- Get API keys from the xAI Console
- Same API features as standalone API access
- Combined billing

## Integrations

### Browser Extensions

Use Grok directly in your browser workflow.

### Slack Integration

Bring Grok into your Slack workspace for team access.

## Pricing

Grok Business has separate pricing from API access:
- Per-seat pricing for team members
- Enterprise pricing for organizations
- Contact sales for custom pricing

## Getting Started

1. Sign up at grok.com
2. Choose your plan (Individual, Team, or Enterprise)
3. Invite team members (Team/Enterprise)
4. Configure settings
5. Start using Grok
