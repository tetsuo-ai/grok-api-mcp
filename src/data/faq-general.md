## About xAI

### What is xAI?

xAI is an AI company that builds Grok, a flagship AI model designed to deliver truthful, insightful answers.

### What is Grok?

Grok is xAI's AI assistant available through:
- The Grok website and apps (consumer)
- The xAI API (developers)

## Products

### What products does xAI offer?

- **Grok**: AI assistant (web, mobile apps)
- **xAI API**: Developer API for integrating Grok
- **Grok Business**: Team/enterprise Grok access
- **Grok Voice**: Real-time voice conversations

### What's the difference between Grok and xAI API?

- **Grok**: Consumer product at grok.com
- **xAI API**: Developer product for building applications

Billing is separate for each.

## Security

### Is xAI affiliated with any cryptocurrency?

**No.** xAI is not affiliated with any cryptocurrency.

We are aware of scam websites unlawfully using the xAI name and logo. Always verify you're on official xAI domains:
- grok.com
- x.ai
- docs.x.ai
- console.x.ai

### How do I report a scam?

If you encounter a scam using xAI's name, please report it to security@x.ai.
