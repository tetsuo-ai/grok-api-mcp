# Grok Business User Guide

Guide for using Grok in a business/team environment.

## Overview

Grok Business provides dedicated workspaces for personal and team use, with enhanced privacy and sharing controls.

## Workspaces

### Personal Workspace

- For individual use
- Available unless disabled by your organization
- Private by default

### Team Workspace

- For collaborative work within your team
- Accessible only with an active license
- Shared with team members

## License Tiers

### SuperGrok

Standard business access with enhanced quotas and features:
- Extended context windows
- Higher rate limits
- Priority access

### SuperGrok Heavy

Upgraded performance for demanding workloads:
- Maximum rate limits
- Highest priority access
- Extended capabilities

## Key Features

### Enhanced Privacy

- Dedicated workspaces separate from consumer Grok
- Data isolation between organizations
- Enterprise-grade security

### Secure Sharing

Share conversations only with team members who have active licenses:
- Share links are only accessible to licensed team members
- If sent to non-team members or unlicensed team members, the link will not open
- Revoke sharing at any time

### Team Collaboration

- Create shared conversation threads
- Collaborate on complex problems
- Share context across team members

## Getting Started

### 1. Accept Invitation

When invited to a Grok Business workspace:
1. Check your email for the invitation
2. Click the invitation link
3. Sign in or create an xAI account
4. Accept the workspace invitation

### 2. Choose Workspace

Select the appropriate workspace for your task:
- **Personal**: For individual work
- **Team**: For collaborative work

### 3. Start Chatting

Begin a conversation with Grok:
- Ask questions
- Request analysis
- Generate content
- Solve problems

## Using Grok

### Basic Interaction

```
You: Analyze this quarterly report and summarize key findings.

Grok: [Provides detailed analysis with key insights]
```

### File Attachments

Upload files for Grok to analyze:
- Documents (PDF, DOCX, etc.)
- Spreadsheets
- Images
- Code files

### Voice Conversations

Use voice mode for hands-free interaction:
1. Click the microphone icon
2. Speak your question
3. Listen to Grok's response

## Sharing Conversations

### Share with Team Members

1. Open the conversation you want to share
2. Click the Share button
3. Select team members or copy the share link
4. Recipients must have active licenses to access

### Sharing Restrictions

- Only licensed team members can access shared conversations
- Shared links do not work for non-team members
- Organization admins can view sharing activity

## Integrations

### Google Drive

Connect Google Drive to search your documents:
1. Go to Settings > Integrations
2. Click Connect Google Drive
3. Authorize xAI to access your Drive
4. Grok will automatically search relevant files

Features:
- Inline citations linking to source files
- Multi-file reasoning
- Hourly sync for changes
- Permission enforcement

### Other Integrations

Check your organization settings for additional integrations configured by your admin.

## Best Practices

### For Productivity

1. **Be specific**: Provide clear, detailed prompts
2. **Use context**: Reference previous messages when relevant
3. **Iterate**: Refine responses through follow-up questions
4. **Organize**: Use separate conversations for different topics

### For Teams

1. **Share thoughtfully**: Only share what's necessary
2. **Document**: Save important insights from conversations
3. **Collaborate**: Use team workspace for group projects
4. **Respect privacy**: Don't share sensitive conversations

## Troubleshooting

### Can't Access Team Workspace

- Verify you have an active license
- Check with your admin about permissions
- Ensure you're signed into the correct account

### Shared Link Not Working

- Recipient must have an active license
- Recipient must be a team member
- Check if sharing was revoked

### Files Not Syncing

- For Google Drive: sync happens hourly
- Check integration is still connected
- Verify file permissions in source app
