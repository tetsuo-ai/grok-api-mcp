# FAQ - xAI Console

Frequently asked questions about the xAI Console.

## Accessing the Console

### How do I access the xAI Console?

Go to https://console.x.ai and sign in with your xAI account.

### What can I do in the Console?

- Manage API keys
- View usage and billing
- Create and manage teams
- Configure settings
- Access Collections
- Manage files

## API Keys

### How do I create an API key?

1. Go to "API Keys" section
2. Click "Create new key"
3. Name your key
4. Set permissions (optional)
5. Copy the key immediately (it won't be shown again)

### How do I revoke an API key?

1. Find the key in the API Keys list
2. Click "Revoke" or the delete icon
3. Confirm revocation

### Can I see my API key again after creation?

No, for security reasons, API keys are only shown once at creation. If you lose it, create a new key.

## Usage & Billing

### Where can I see my usage?

Go to "Usage" or "Usage Explorer" in the Console sidebar.

### How do I add credits?

1. Go to "Billing"
2. Click "Add Credits"
3. Select amount
4. Complete payment

### Where are my invoices?

Go to "Billing" → "Invoices" to view and download past invoices.

## Teams

### How do I switch teams?

Use the team selector in the top navigation or sidebar to switch between teams.

### How do I invite someone to my team?

1. Go to "Team" or "Members"
2. Click "Invite"
3. Enter email address
4. Select role
5. Send invitation

## Collections

### How do I create a Collection?

1. Go to "Collections"
2. Click "Create Collection"
3. Name your collection
4. Start uploading documents

### How do I upload documents?

1. Open a collection
2. Click "Upload" or drag and drop files
3. Wait for processing to complete

## Settings

### How do I change my account settings?

Click your profile icon in the top right, then select "Settings" or "Account Settings".

### How do I configure notifications?

Go to Settings → Notifications to configure email and alert preferences.

## Models

### How do I see available models?

Go to "Models" to see all available models with their capabilities and pricing.

### How do I check my rate limits?

Your team's rate limits are shown on the Models Page in the Console.

## Troubleshooting

### The Console isn't loading

- Clear your browser cache
- Try a different browser
- Check https://status.x.ai for outages
- Disable browser extensions

### I can't create an API key

- Verify you have the necessary permissions
- Check if you've reached the key limit
- Contact support if the issue persists

### My usage data isn't updating

- Usage data may have a slight delay (up to a few minutes)
- Try refreshing the page
- Check the date range filter

## Mobile Access

### Is there a mobile app?

The Console is web-based and accessible from mobile browsers, though optimized for desktop.
