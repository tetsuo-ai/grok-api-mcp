# FAQ - Grok Website / Apps

While the documentation is mainly meant for API users, here are commonly asked questions for consumer-facing Grok website and apps.

## Account

### How do I create a Grok account?

Visit grok.com and sign up with:
- Email address
- X (Twitter) account
- Google account

### Is the Grok account the same as xAI API account?

Yes, the account is shared between Grok and xAI API, but **billing is separate**.

### How do I connect my X account?

1. Go to Grok Website Settings
2. Click "Connect X Account"
3. You'll be taken to X's SSO page
4. Authorize the connection
5. Your X account becomes a sign-in method

## Subscription

### What Grok plans are available?

- **Free**: Basic access with limits
- **SuperGrok**: Enhanced access and features
- **Grok Business**: Team features
- **Grok Enterprise**: Organization features

### Does X Premium include Grok?

X Premium may include Grok access, but this is separate from xAI API. X Premium purchases won't affect your xAI API service status.

### How do I upgrade my plan?

Go to grok.com → Settings → Subscription and select your desired plan.

## Features

### What can Grok do?

- Answer questions
- Analyze images
- Search the web
- Search X posts
- Generate images
- Code assistance
- File analysis

### Can Grok access the internet?

Yes, Grok has built-in web search and X search capabilities for real-time information.

### Can Grok see images?

Yes, you can upload images or share URLs for Grok to analyze.

## Privacy

### Are my conversations private?

Your conversations are private to your account. Review the privacy policy for full details.

### Does Grok learn from my conversations?

Review xAI's privacy policy for information about data usage.

### How do I delete my conversation history?

Go to Settings → Privacy → Delete History.

## Troubleshooting

### Grok isn't responding

- Check your internet connection
- Try refreshing the page
- Check status.x.ai for outages
- Clear browser cache

### I'm getting rate limited

Free tier has usage limits. Consider upgrading for higher limits.

### Image upload isn't working

- Check file size (max limit applies)
- Supported formats: JPG, PNG
- Try a different browser

## Security

### Is Grok safe to use for sensitive topics?

Grok is designed for general assistance. Avoid sharing highly sensitive personal information.

### Is Grok affiliated with any cryptocurrency?

**No.** xAI is not affiliated with any cryptocurrency. Be aware of scam websites unlawfully using the xAI name and logo.
