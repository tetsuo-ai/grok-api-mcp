# FAQ - xAI API Accounts

Frequently asked questions about xAI API accounts.

## Creating an Account

### How do I create an xAI account?

You can create an account at:
- https://accounts.x.ai
- https://console.x.ai

### Can I sign up with my X account?

Yes! To link your X account automatically to your xAI account, choose to sign up with X account during registration.

### Is the account shared between Grok and xAI API?

Yes, the account is shared between Grok and xAI API, but **billing is separate** for Grok and xAI API.

## Account Management

### How do I reset my password?

1. Go to the login page
2. Click "Forgot Password"
3. Enter your email
4. Check your email for reset instructions
5. Create a new password

### How do I change my email address?

1. Log in to console.x.ai
2. Go to Account Settings
3. Update your email
4. Verify the new email address

### How do I delete my account?

Contact support@x.ai to request account deletion. Note that:
- All API keys will be revoked
- Usage history will be retained for billing purposes
- Active subscriptions will be cancelled

## Authentication

### What authentication methods are supported?

- Email and password
- Sign in with X
- SSO (Enterprise only)

### How do I enable two-factor authentication?

1. Go to Account Settings → Security
2. Click "Enable 2FA"
3. Scan the QR code with your authenticator app
4. Enter the verification code
5. Save backup codes

## API Keys

### Where do I get my API key?

1. Log in to console.x.ai
2. Navigate to "API Keys"
3. Click "Create new key"
4. Copy and save your key securely

### Can I have multiple API keys?

Yes, you can create multiple API keys for different purposes (development, production, testing).

### What happens if my API key is compromised?

1. Immediately revoke the compromised key
2. Create a new key
3. Update your applications
4. Review usage logs for unauthorized access

## Linking Accounts

### Can I link my X account after creating an account?

Yes, go to Account Settings → Linked Accounts and connect your X account.

### What are the benefits of linking my X account?

- Easier sign-in
- Access to X search features in API
- Unified account management

## Teams

### How do I create a team?

1. Go to console.x.ai
2. Click "Create Team"
3. Enter team name
4. Invite members

### Can I be part of multiple teams?

Yes, you can belong to multiple teams with different roles in each.

### How do I leave a team?

1. Go to the team's settings
2. Click "Leave Team"
3. Confirm

**Note**: Team owners cannot leave without transferring ownership.

## Support
