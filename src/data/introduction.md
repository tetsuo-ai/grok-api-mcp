# Introduction

Welcome to the xAI API documentation.

## Getting Started

To get started building with the xAI API, please head to [The Hitchhiker's Guide to Grok](./tutorial.md).

## xAI API vs Grok Website

The xAI API and Grok website/apps are **separate offerings**:

- Your purchase on X (e.g., X Premium) won't affect your service status on xAI API
- Your xAI API subscription won't affect your X Premium status
- Billing is separate for each service

## What is Grok?

Grok is xAI's flagship AI model designed to deliver truthful, insightful answers. It powers both:
- The Grok website and apps (consumer-facing)
- The xAI API (developer-facing)

## API Capabilities

The xAI API provides:

### Text Generation
- Chat completions
- Responses API
- Streaming support
- Multi-turn conversations

### Reasoning
- Advanced reasoning models
- Step-by-step problem solving
- Math and logic capabilities

### Vision
- Image understanding
- Image generation
- Image editing

### Video
- Video generation
- Video editing
- Video understanding in search

### Voice
- Real-time voice conversations
- WebSocket-based Voice Agent API
- Multi-language support

### Tools & Agents
- Function calling
- Built-in web search
- X (Twitter) search
- Code execution
- Document search (Collections)

### Enterprise Features
- Grok Business
- Organization management
- SSO integration
- Audit logging
